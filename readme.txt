==== SISContactInformation to save the form information to database ====

Tags: contact form, save contact information, database
Version: 1.0
Tested only in : 3.4.2

It will save the information entered by user to mysql database and the information will displayed in admin. 
Admin will receive an email of submitted information

Installation:

Use this below short code in page or post after installed and activated the plugin, then you can see the contact form
in particular page or post

[siscontactform]
